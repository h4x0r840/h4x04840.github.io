无线安全

无线安全的方向

1. 无线数据报文窃听
    监听wifi，嗅探蓝牙等
2. 无线信号重放攻击
    汽车钥匙，可以使用时间戳或者随机性等防止重放机制
3. 无线信号欺骗攻击
    掌握了报文格式和密钥等就可以构造
4. 无线信号劫持攻击
    手机信号干扰，导致手机回退使用2G信号，然后使用伪基站

学习思路

1. 无线通信协议和认证方式等
2. 电子与无线信号相关知识
3. 动手能力，焊接硬件等
4. 良好的计算机基础
5. 熟练使用GNU Radio等软件

RFID智能卡的常见破解知识

125kHZ 低频ID卡

1. 门禁卡、考勤卡等 
2. UID也是只读的，但是可以买到UID白卡，导致克隆UID攻击 
3. Proxmark 直接模拟UID 
4. 360 HackID

Mifare Classic 4K卡

1. 4个扇区，每个扇区32个区块 + 16个扇区，每个扇区8个区块，共256个区块 
2. 第一个区块有UID，理论上是只读的 
3. 每个扇区的最后一个区块保存了Key和访问控制条件 
4. 扇区其他部分加密存储了数据，比如金额，最后消费时间，客户编号等 
5. 验证漏洞，已知一个扇区密钥可能导致破解 
6. 怎么防？数据联网验证，实时或非实时对账

车库门/车门无线钥匙破解

车库门钥匙的原理

1. 固定码技术，导致密钥空间是有限的 
2. 12个拨码开关就只有4096种组合 
3. 为了增加稳定性，相同信号会被连续发送，每个bit之间也有间隔 
4. 破解的时候，舍弃上面的条件可以大大的加快速度

使用的工具

OpenSesame

车门钥匙的原理

1. 滚动码，每次发射的信号都是不一样的，钥匙和汽车的滚动码各自存储和同步 
2. 密钥空间大大增大 
3. 可以使用隔离录制重放攻击，但是需要接触车钥匙  
4. 使用的工具 
5. 电视棒，录制工具 
6. RfCat，播放工具

蓝牙嗅探工具

Ubertooth

1. 2010年出现，无线安全黑客Michael Ossmann 
2. 可以用过频谱分析，监听LAP地址等，得知附近的蓝牙设备 
3. Kismet插件，分析蓝牙数据包

BLE Sinffer

1. 用于低功耗蓝牙，手环等 
2. RfCat，播放工具 
3. http://www.freebuf.com/news/88281.html 如何调戏别人的手环

移动通信网络安全

GSM主动攻击

1. 主动攻击，攻击者伪装为基站，发送洗脑，诱导用户连接 
2. GSM单向鉴权，无法对基站鉴权 
3. 可以导致通话、短信和网络流量被窃听 
4. 硬件USRP，软件OpenBTS等

GSM被动攻击

1. 接收机接收与用户相同的基站发送出来的信号，再进行分析和解码 
2. OsmocomBB

WiFi安全

破解WiFi密码

1. WEP / WPA / WPA2-PSK等认证方式 
2. WEP等认证方式缺陷较多，已经基本不再使用 
3. WPA2-PSK使用字典PSK生成PMK，然后结合其他信息生成PTK，再加上原始报文生成MIC，与AP的MIC作比较，如果一致，PSK就是密钥 
4. WPS暴力破解 
5. aircrack系列软件可以用于破解WiFi

3. http://www.freebuf.com/author/radiowar 
4. https://en.wikipedia.org/wiki/Wi-Fi_Protected_Access

Python-shodan编写基础的查询脚本

----
import shodan
import sys

# Configuration
API_KEY = "YOUR_API_KEY"

#Input validation
if len(sys.argv) == 1:
    print 'Usage: %s <search query>' % sys.argv[0]
    sys.exit(1)

try:

    # Setup the api
    api = shodan.Shodan(API_KEY)

    # Perform the search
    query = ' '.join(sys.argv[1:])
    result = api.search(query)

    # Loop through the matches and print each IP
    for service in result['matches']:
        print service['ip_str']
except Exception as e:
    print 'Error: %s' % e
    sys.exit(1)
----
